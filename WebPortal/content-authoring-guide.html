TileStorm Content Authoring Guide

This is a short placeholder guide for creators and submitters. It covers the basics only so the portal stays simple.

1. Create Content

  • Build a map in TileStorm and keep the final published version clean.
  • Use the in-game publish flow to generate the export package.
  • The published map file stays in `maps/`; the preview image stays in `thumbs/`.

2. Submit Content

  • Open the shared map panel in a build with publish access.
  • Press Publish Current.
  • The upload will send the map package and a matching thumbnail separately using the same hash name.

3. Download Content

  • Open the portal landing page.
  • Pick a map and open its map file link or import it through the game.
  • The thumbnail is just a preview asset and is not part of the map export.

4. Notes

  • Delete by hash removes the map file and the matching thumbnail.
  • Keep thumbnails small and readable.
  • This guide is intentionally not exhaustive.
