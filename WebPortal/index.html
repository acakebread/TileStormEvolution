Quick Notes

  • Thumbnails are stored separately as `thumbs/<mapHash>.png`.
  • Map packages stay in `maps/` and are not bundled with the PNG.
  • Publish updates both files, delete removes both by hash.

README

This site is the public read layer for shared TileStorm content. The game publishes map packages and matching thumbnails into this repository, and this page reads the manifest to show what is available.

Use the authoring guide for the longer workflow notes. Keep the README compact; this page is intended to stay lightweight and easy to scan.
